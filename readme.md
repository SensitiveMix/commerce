# e-commerce-platform

## Installation
    git clone https://bitbucket.org/wechatpla/master.git


## Install dependencies:
    npm install

## Start the server:

*  nodemon app.js
*  node ./bin/www
*  nodemon ./bin/www
*  npm start

## run test

* npm test

## run test-cov

* npm run-script test-cov
