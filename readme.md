## Installation
    git clone https://bitbucket.org/wechatpla/master.git


## Install dependencies:
    npm install

## Start the server:

```bash

dev serve       : npm run dev               

```

```bash

browser serve   : npm run browsersync   

```

```bash

product serve   : npm run production   

```

```bash

manual package  : npm run build

```
